## What is this?

The file management system lets you upload, store, and view all documents attached to a patient — lab reports, X-rays, consent forms, insurance documents, discharge summaries, and any other medical files. Files are organized in the patient's detail view under the "Files" tab and are securely stored. You can upload new files via the paperclip icon, download files to your device, and see who uploaded each file and when.

## How to use it

1. Open a patient's detail view by tapping their card from the patient list.
2. Tap the "Files" tab at the bottom of the screen.
3. To upload a new file, tap the paperclip icon (upload button) at the top of the Files tab.
4. Select a file from your device (photos, PDFs, documents, images are all supported).
5. Add an optional label or note describing the file (e.g., "Consent form signed", "CT scan 2-Apr").
6. Tap "Upload" — the file will be stored and linked to this patient.
7. To view a file, tap it in the list — it will open in a viewer or download automatically depending on file type.
8. To download a file to your device, tap the download icon next to the filename.
9. To delete a file (if you have permission), tap the trash icon and confirm.

## Common questions

**Q: What file types can I upload?**
A: Most common formats work: PDF, JPG, PNG, Word documents (.docx), Excel spreadsheets (.xlsx), and plain text. If a file type is not supported, you will see an error message when trying to upload.

**Q: How large can a file be?**
A: Files up to 50 MB are supported. If your file is larger, try compressing it or splitting it into smaller parts.

**Q: Can I upload multiple files at once?**
A: Currently, you upload one file at a time. To upload multiple files quickly, open the Files tab and repeat the upload process for each file.

**Q: Who can see the files I upload?**
A: All staff members who have access to this patient's record can view the files. Files are never shared with the patient unless you explicitly send them through a patient communication channel.

**Q: Can I edit or update a file I already uploaded?**
A: No, files cannot be edited once uploaded. If you need to replace a file, delete the old one and upload the new version with an updated label.

## Troubleshooting

**Problem: I try to upload a file but get an error message "File type not supported".**
Solution: The file type you chose may not be supported. Try converting the file to a different format (e.g., convert an image to JPG, or a document to PDF) and upload again.

**Problem: The upload is very slow or keeps failing.**
Solution: Check your internet connection — slow connections may cause timeouts. Try uploading a smaller file first to test. If the problem persists, close the app and reopen it, then try again.

**Problem: I can see a file was uploaded but when I tap it, it won't open or download.**
Solution: Some file types may not open in the app viewer but can still be downloaded. Tap the download icon instead to save the file to your device and open it with your computer or phone's file app. If the download also fails, contact support.
